# 📦 Експорт префаба в інший проєкт Unity (через .unitypackage)

Детальний гайд зі всіма кроками та типовими помилками.

[<< Назад до вимог](index.md)

---

## 🔧 Що це за спосіб

- Використовується стандартна функція Unity
- Створюється файл `.unitypackage`
- Він містить:
  - префаб
  - всі залежності (якщо увімкнено)

---

## 🚀 Покроковий процес експорту

### 1. Відкрий проєкт-джерело

- Запусти Unity
- Відкрий проєкт, де знаходиться префаб

---

### 2. Знайди префаб

- Перейди у вікно **Project**
- Знайди файл `.prefab`
- Переконайся:
  - немає помилок у Console
  - всі матеріали підключені
  - немає Missing-компонентів

---

### 3. Запусти експорт

- Клікни **ПКМ по префабу**
- Обери: `Export Package...`

або:

- Меню: `Assets → Export Package...`

---

### 4. Увімкни залежності

- У вікні постав галочку:
  - ✅ **Include dependencies**

---

### 5. Перевір список

Зазвичай у списку будуть:

- префаб
- матеріали
- текстури
- моделі (FBX)
- анімації
- вкладені префаби
- скрипти

📌 Не знімай галочки без потреби

---

### 6. Експорт

- Натисни **Export**
- Збережи файл `.unitypackage`

---

## 📥 Імпорт у інший проєкт

### 1. Відкрий новий проєкт

---

### 2. Імпорт пакета

- Меню: `Assets → Import Package → Custom Package...`
- Вибери `.unitypackage`

---

### 3. Підтвердження

- Залиш усе відміченим
- Натисни **Import**

---

## ⚠️ Типові помилки

### ❌ 1. Забули `Include dependencies`

**Симптоми:**

- рожеві матеріали
- відсутні текстури
- зламаний вигляд

**Причина:**

- експортувався тільки `.prefab`

**Рішення:**

- повторити експорт з галочкою

---

### ❌ 2. Missing Script

**Симптоми:**

- `Missing (Mono Script)`

**Причина:**

- скрипт не скомпілювався
- інший namespace
- відсутній пакет

**Рішення:**

- перевір Console
- перевір назву класу = назві файлу
- встанови потрібні пакети

---

### ❌ 3. Рожеві матеріали

**Симптоми:**

- об’єкт повністю рожевий

**Причина:**

- інший Render Pipeline (URP / HDRP / Built-in)

**Рішення:**

- встановити той самий pipeline
- або змінити shader

---

### ❌ 4. Префаб виглядає інакше

**Симптоми:**

- інший масштаб
- інше освітлення
- UI "з'їхав"

**Причина:**

- не перенесені Project Settings

**Рішення:**

- перевір:
  - Canvas Scaler
  - Lighting
  - Quality Settings
  - Sorting Layers

---

### ❌ 5. Вкладені префаби зникли

**Симптоми:**

- частини об’єкта відсутні

**Причина:**

- не всі залежності включені

**Рішення:**

- не знімати галочки
- експортувати знову

---

### ❌ 6. Конфлікт файлів

**Симптоми:**

- Unity пропонує заміну файлів

**Причина:**

- однакові шляхи в Assets

**Рішення:**

- використовуй окрему папку:
  - `Assets/Export/MyPrefab`

---

### ❌ 7. Не працюють сторонні пакети

**Симптоми:**

- функціонал відсутній

**Причина:**

- package не включає UPM залежності

**Рішення:**

- перевір:
  - Package Manager
  - manifest.json

---

### ❌ 8. Різні версії Unity

**Симптоми:**

- помилки після імпорту
- дивна поведінка

**Причина:**

- несумісність версій

**Рішення:**

- використовуй однакову версію Unity

---

### ❌ 9. Занадто великий package

**Симптоми:**

- файл займає багато місця

**Причина:**

- зайві залежності

**Рішення:**

- ізолюй префаб у окрему папку

---

## 🧠 Best Practice

Перед експортом:

- створи окрему папку:
  - `Assets/MyPrefabPack`
- перемісти туди:
  - префаб
  - матеріали
  - текстури
- переконайся, що:
  - немає зайвих залежностей
