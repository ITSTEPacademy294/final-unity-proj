# 🎮 Final Project — Coin Runner: Infinite World (Unity)

## 🧠 Опис гри
Вам потрібно створити просту 3D-гру з нескінченним процедурно згенерованим світом.  
Гравець рухається по світу, збирає монетки, і залежно від їх кількості з ним відбуваються різні зміни ("прикольні ефекти").

---

## 🎯 Основна ідея
- Світ поділений на чанки (chunks) — квадратні ділянки (наприклад 10x10)
- Чанки створюються навколо гравця
- Чанки видаляються, якщо гравець відходить далеко
- На чанках з’являються монетки
- Гравець збирає монети → змінюється його стан

---

## 🧩 Основні механіки

### 🧍 Гравець
- Рух (WASD або стрілки)
- Камера слідує за гравцем

### 🪙 Монетки
- Спавняться випадково в чанках
- При зіткненні:
  - зникають
  - додають до рахунку

### 🌍 Генерація світу
- Світ нескінченний
- Генерується навколо гравця
- Старі чанки автоматично видаляються

### ⚡ Ефекти (обов'язково хоча б 1)
Приклади:
- 5 монет → гравець стає більшим
- 10 монет → швидкість збільшується
- 15 монет → змінюється колір
- 20 монет → будь-який інший ефект
